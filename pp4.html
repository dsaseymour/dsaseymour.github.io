Project Completed

Live Site Github Documentation

CHALLENGES:

The challenge this project addressed was that I needed a platform to display my coding projects, and any other content relevant to my work.

SOLUTION:

Taking advantage of my web development skills gave me a level of freedom I may have not been able to gain with web building platform like wix or squarespace. I chose to have a static site because I favoured I having a simple, quick site and there was no significant advantage to creating a dynamic site as opposed to static for content that I have no problem manually updating.

OUTCOME:

I have a website that provides links to my social media, github projects and provides visitors a means to contact me and I am much more readily accessible to the public.
